Docker Compose files for the OAI CN5G (5G Core Network).

Using the latest version (v2.1.10) – (2 months old)

To run the Core Network, execute the following command:

    docker-compose up -d

or:

    docker-compose -f ./docker-compose-lmf.yaml up -d

Both docker-compose.yaml and docker-compose-lmf.yaml files are identical.

Also uses MongoDB instead of MySQL (benefits are uncertain).
